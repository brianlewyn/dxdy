¿Por qué dxdy es una App Web y no de otro tipo?

La App dxdy se hizo como una App Web principalmente por que el usuario no debe intalar nada, sino tan solo buscar el link brianlewyn.github.io/dxdy o el nombre dxdy para poder ingresar app, esto lo hace a través del navegador de su de propio smartphone, tablet o computadora.


¿Qué is html, css y js?

Estos tres nombres se refieren a tres tecnologías web, que se usan a para programar paginas, sitios o app web, son el leguaje que solo entiende el navegador, ya que hay leguajes como Python, C++, C#, entre otros que se visualizan y se ejecutan de una manera muy distinta.

HTML es un leguaje unicamente para programar todo el esqueleto de una pagina que vendrían a ser principalmente los titulos, parrafos, fraces. Todo esto se encierra en etiquetas especiales para formar una jeraquía, algo similar a la jeraquía de operaciones en matemáticas.

CSS es un lenguaje unicamente para programar todo el diseño de la página, esto vendría ser como la piel del esqueleto, con este leguaje se puede dar colores, formas, tamaños, transiciones, etc; todo esto a través de nombres de clase. También se uso Sass, pero solo una manera más comoda y bonita de escribir el codigo de los css.

JS o JavaScript se usa solo para programar acciones, hacer calculos, entre otros; esto vendría ser como el corazón, lo que le da vida a dicho esqueleto.


¿Qué es Bootstrap?

Bootstrap es un framework, un entorno de trabajo el cual tiene un conjunto estandarizado de conceptos, prácticas y criterios para enfocar un tipo de problemática particular que sirve como referencia, para enfrentar y resolver nuevos problemas de índole similar.

Es por ello que se uso especialmente Bootstrap para darle a la App un mejor acabado en cuanto al diseño, siguiendo un reglas ya estandarizadas para que la App se más estable a la hora de adaptaser a un tamaño especifico de dispositivo. Cabe resaltar, que este framework usa los lenguajes anteriores para formular la guías a seguir.


¿Qué es Go (Golang)?

Golang es un lenguaje de programación creado por Google, es un lenguaje de robusto como Java y c++, pero que asu vez es tan dinámico como Python y JavaScript, combina lo mejor de los leguajes de programación compilados y los de tipo interpretado, todo esto para ser el más rápido dentro del ecosistema en servidores. Se uso GO para desarrollar toda la lógica de las derivadas. Este y JavaScript se conectan a través de peticiones y respuestas del cliente (dispositivo) y servidor para enviar los datos de ida y de vuelata.


¿Qué es Figma?

Es una App de escritorio y también web que fácilita el desarrollar de diseños para paginas, sitios y App, todo esto para ahorrar mayor tiempo y tener una idea más aproximada del produto final. Se uso Figma justamente para desarrollar el diseño, por ejemplo coo iba ser la estructura, los colores o que tipo iconos iba llevar la App, con que efectos.


¿Qué es Git y GitHub?

Git es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

En cambio, GitHub es una un servico de nube especial para alojar proyectos, utilizando el sistema de control de versiones Git.

Se usaron esta dos tecnologías para tener un mayor control a la hora de desarrollar el código de toda la App.


¿Posibles problemas o inconvenintes de la App?

El primer inconveninte podría ser el peso con la que va y viene la información, pero como la App es bastante simple el peso es de algunos cuantos Mb (mega bytes) nada que ningun intenert no pueda hacer. El segundo y el más importante es que la App no funciona sino tiene internet, pero seamos sinceros casi todos tenemos intenert o un plan de datos, si alguien quiere usar la app esto lo debe tener en cuenta.
